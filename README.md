# Student Management System1 v2.0

## Project Overview

The Student Management System is a web-based application developed using Flask and SQLite. It helps manage student attendance and academic results through separate Admin and Student dashboards.

## Features

* Admin Login
* Student Login
* Attendance Management
* Result Management
* Edit and Delete Records
* Percentage and Grade Calculation
* Logout Functionality
* Responsive User Interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Jinja2

## How to Run

1. Install Flask

   pip install flask

2. Run the application

   py app.py

3. Open the browser and visit:

   http://127.0.0.1:5000
